# 🚀 AI Study Time Predictor

An AI-powered web application that predicts the optimal study time required for a student based on behavioral and environmental factors such as focus level, sleep, and subject difficulty.

---

## 🧠 Features

- 📊 Predicts required study time using Machine Learning
- 🎯 Personalized recommendations based on user input
- 🌐 Simple and clean web interface
- ⚡ Fast Flask API backend
- 📁 Synthetic dataset with realistic logic

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Python (Flask)  
- **Machine Learning:** Scikit-learn  
- **Model Storage:** Joblib  

---


### 5. Open frontend
Open `frontend/index.html` in your browser

---

## 📊 Input Parameters

- Hours Studied  
- Focus Level (1–10)  
- Sleep Hours  
- Difficulty (1–5)  

---

## 🎯 Output

- Predicted study time required (in hours)

---

## ⚠️ Limitations

- Uses synthetic dataset  
- Predictions are approximations  
- Not trained on real-world user data
